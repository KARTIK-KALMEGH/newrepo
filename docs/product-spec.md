# Product Spec (Condensed)

## Personas
- Student (primary)
- Mentor/Trainer
- Recruiter

## Core User Journey
1. Onboard and choose target career role.
2. Complete assessment.
3. Receive AI skill-gap report.
4. Follow weekly roadmap.
5. Submit projects and apply to internships.
6. Improve resume and profile score.

## Success Metrics
- D7 retention > 35%
- Assessment completion > 70%
- Internship application rate > 40%
- Interview shortlist uplift per cohort
