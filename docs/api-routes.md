# API Routes

## Auth
- POST `/api/auth/register`
- POST `/api/auth/login`
- GET `/api/me`

## Assessment
- POST `/api/assessment/submit`
- GET `/api/assessment/report/:userId`

## AI
- POST `/api/ai/mentor`
- POST `/api/roadmap/generate`

## Internships
- GET `/api/internships`
- POST `/api/internships/:id/apply`

## Resume
- POST `/api/resume/build`
- GET `/api/resume/:userId`
